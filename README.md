# 🚗 Car Price Prediction using Machine Learning

A complete end-to-end Machine Learning project that predicts the **selling price of a used car** based on various features such as fuel type, kilometers driven, ownership, and more.

---

## 📌 Project Overview

In the used car market, determining the correct selling price is crucial for both buyers and sellers. This project uses a **Random Forest Regression model** to estimate the resale value of cars with high accuracy.

The application is deployed using **Streamlit**, allowing users to interactively input car details and get instant predictions.

---

## 🎯 Features

- Predict car selling price in real-time
- User-friendly web interface using Streamlit
- Data preprocessing and feature engineering
- Machine Learning model training and evaluation
- Model deployment ready

---

## 🧠 Machine Learning Workflow

1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Feature Engineering (Car Age)  
4. Encoding Categorical Variables  
5. Feature Scaling  
6. Model Training (Random Forest Regressor)  
7. Model Evaluation (MSE, RMSE, R² Score)  
8. Deployment using Streamlit  

---

## 📊 Input Features

- Present Price (Showroom Price)
- Kilometers Driven
- Number of Previous Owners
- Fuel Type (Petrol/Diesel/CNG)
- Seller Type (Dealer/Individual)
- Transmission Type (Manual/Automatic)
- Year of Purchase (used to calculate Car Age)
